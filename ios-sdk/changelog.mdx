## 升级指南

### 从 v0.9.1 升级到 v0.9.2

#### 自动升级

如果您使用 Swift Package Manager：

1. 在 Xcode 中选择 `File` → `Add Packages...`
2. 找到 SPAvatarKit 包
3. 点击更新按钮

如果您使用 CocoaPods：

```bash
pod update SPAvatarKit
```

#### 手动升级

1. 更新依赖版本
2. 清理项目缓存
3. 重新编译项目
4. 测试新功能

#### 破坏性变更

v0.9.2 没有破坏性变更，可以直接升级。

### 从 v0.9.0 升级到 v0.9.1

#### 新功能使用

```swift
// 新增的网络状态监听
let networkMonitor = NWPathMonitor()
networkMonitor.pathUpdateHandler = { path in
    if path.status == .satisfied {
        // 网络可用
    } else {
        // 网络不可用
    }
}

// 新增的性能监控
AvatarSDK.enablePerformanceMonitoring(true)
```

## 已知问题

### v0.9.2 已知问题

1. **内存使用**: 在某些低端设备上内存使用可能较高
   - **状态**: 已确认，计划在 v0.9.3 中修复
   - **临时解决方案**: 使用低质量渲染模式

2. **网络重连**: 在网络切换时可能出现短暂卡顿
   - **状态**: 已确认，计划在 v0.9.3 中优化
   - **临时解决方案**: 手动重新初始化渲染器

### v0.9.1 已知问题

1. **权限检查**: 在某些情况下权限检查可能不准确
   - **状态**: 已在 v0.9.2 中修复
